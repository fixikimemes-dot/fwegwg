# Telegram-бот для контроля успеваемости за день

Я собрал MVP-бота под твой запрос: он сам пишет в Telegram, напоминает утром/днём/вечером, задаёт вопросы про биографию и увлечения, собирает план кнопками, показывает его как таблицу и делает AI-разбор по твоему профилю и результатам.

## Что умеет бот

- спрашивает биографию, увлечения, часовой пояс и время напоминаний;
- сам пишет тебе по расписанию;
- добавляет задачи через кнопки;
- показывает план на сегодня в виде таблицы прямо в Telegram;
- позволяет быстро отметить задачу как выполненную, перенесённую или удалить её;
- просит промежуточный отчёт днём и вечером;
- делает AI-анализ дня;
- даёт ответы в режиме `AI-коуч`;
- экспортирует историю за последние 30 дней в ZIP с CSV-файлами.

## Стек

- Python
- `python-telegram-bot`
- `JobQueue` для напоминаний
- `SQLite` для хранения данных
- `OpenAI Responses API` для AI-анализа

## Как запустить

### 1. Создай Telegram-бота

- открой `@BotFather`;
- выполни `/newbot`;
- возьми токен;
- вставь токен в `.env`.

### 2. Подготовь окружение

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

### 3. Заполни `.env`

Минимум нужно указать:

```env
TELEGRAM_BOT_TOKEN=...
OPENAI_API_KEY=...
```

Если `OPENAI_API_KEY` не задан, бот всё равно запустится, но AI будет работать в упрощённом локальном режиме без обращения к модели.

### 4. Запусти

```bash
python -m bot.main
```

## Docker

```bash
docker build -t study-tracker-bot .
docker run --env-file .env study-tracker-bot
```

## Как пользоваться

1. Открой бота в Telegram.
2. Нажми `/start`.
3. Ответь на вопросы про себя.
4. Добавь задачи кнопкой `➕ Добавить задачу`.
5. Получай напоминания и отмечай выполнение.
6. Нажимай `🧠 AI-анализ`, чтобы получить разбор дня.

## Важное

- Бот сможет писать тебе сам только после того, как ты один раз откроешь чат и нажмёшь `/start`.
- Чтобы напоминания работали, процесс бота должен быть постоянно запущен на сервере, VPS или другом хостинге.
- Вся локальная история хранится в SQLite-файле, путь настраивается через `DB_PATH`.
